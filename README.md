# 小象棋 PWA

一款给小朋友在 iPad 上离线玩的中国象棋 PWA。第一版是纯前端静态应用，不需要后端、不需要局域网服务。

## 功能

- 人机对弈，玩家默认执红先行。
- 三档难度：入门、初级、进阶。
- 点击棋子高亮合法走法。
- 非法走法提示、悔棋、提示一步、教学提示开关。
- Safari 添加到主屏幕后可离线打开。

## 本地开发

```bash
npm install
npm run dev
```

开发或预览时请看终端输出的实际地址。如果提示端口被占用，Vite 会自动换到下一个端口，例如 `http://localhost:4176/`。不要手动假设一定是 `4173`。

## 验证

```bash
npm test -- --run
npm run build
```

## 静态部署

构建产物在 `dist/`，可以部署到 GitHub Pages、Vercel 或 Netlify。部署完成后需要 HTTPS 地址，iPad Safari 打开站点后选择“添加到主屏幕”。首次加载完成后，PWA 会缓存静态资源，后续可离线玩。

## iPad 缓存排查

如果更新后发现“对方不走棋”或界面不像最新版本，先确认打开的是小象棋站点，不是其他本地项目端口。再关闭已添加到主屏幕的旧图标，重新用 Safari 打开最新 HTTPS 地址；必要时删除主屏幕旧图标后重新添加。PWA 会使用 service worker 缓存静态资源，旧缓存可能让 iPad 继续显示旧版本。

## 已知边界

- 第一版不包含联网对战、账号、排行榜、开局库和残局关卡。
- AI 是本地轻量搜索，目标是儿童练习，不是专业象棋引擎。
